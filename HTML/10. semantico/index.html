Etiquetas semanticas

Las etiquetas semánticas son aquellas que aportan significado al contenido de una página web. A continuación se muestra una lista de algunas de las etiquetas semánticas más comunes:

  • <header>: Esta etiqueta se utiliza para definir la cabecera de una página web.
  • <nav>: Esta etiqueta se utiliza para definir la navegación de una página web.
  • <main>: Esta etiqueta se utiliza para definir el contenido principal de una página web.
  • <section>: Esta etiqueta se utiliza para definir secciones de contenido relacionadas.
  • <article>: Esta etiqueta se utiliza para definir un artículo independiente dentro de una página web.
  • <aside>: Esta etiqueta se utiliza para definir contenido relacionado pero independiente del contenido principal de una página web.
  • <footer>: Esta etiqueta se utiliza para definir el pie de una página web.
