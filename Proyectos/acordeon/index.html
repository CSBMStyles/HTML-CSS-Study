01 ¿Presenta persistencia?
La persistencia es la capacidad de un objeto de mantener su estado a lo largo del tiempo.
02 ¿SQL inyection afecta?
El SQL inyection es una técnica de hacking que consiste en insertar código SQL en una consulta que se ejecuta en la base de datos.
03 ¿Es rentable?
La rentabilidad es la capacidad de un objeto de generar beneficios económicos.
